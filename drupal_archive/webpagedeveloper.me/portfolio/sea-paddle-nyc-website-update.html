SEA Paddle NYC - Website update

Submitted by David on Sat, 2010-05-22 14:38
  • drupal
  • portfolio
Client:

Kraus Creative

Work: Drupal (views, cck, views_cycle, views_carousel), Version Update
Date: May, 2010
Link: SEA Paddle NYC

Kraus Creative wanted to update the previous SEA Paddle NYC website so that the client could more easily maintain the site. Kraus Creative already had a design ready so only the implementation of the design was needed. Jacob implemented the Drupal theme for SEA Paddle NYC brilliantly and all that was left to do was to implement the new features. The new features included a slideshow on the front page, a sponsor carousel on the front page, an area to embed Flickr sets, an area to embed Youtube videos and a blog / news area.

The slideshow was implement using the views cycle module which allowed for transitioning images on the front page. The system was setup using CCK, imagefield and imagefield_crop so that the end user could easily upload, reorganize and crop image that should appear on the front page.

The sponsor carousel was created using the views_carousel module. Also, using CCK, imagefield and views the client is able to easily manage all of the sponsors. The sponsors also integrate with the sponsors page.

The Flickr sets used a CCK and contemplate allow the user to enter a Flickr set id only so that a Flikr set slideshow would display on the website.

The blog and news area again used the standard CCK content type.

I think this website came together very well.  Sevearl aspects of the site came out looking better and are much easier to manage from an end user perspective.  It was a pleasure working with Kraus Creative on a complete website project for the first time and I'm looking forward to the possiblity next one.  Thanks Roman!
